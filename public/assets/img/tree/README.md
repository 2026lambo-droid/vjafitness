## Image Specifications

- **Format**: JPG recommended
- **Hero images**: Landscape orientation, high resolution (1920x1080 or larger)
- **Service images**: Square or landscape (800x600 minimum)
- **Gallery images**: Various sizes (see code for specific heights)
- **Optimization**: Compress images for web (aim for <500KB each)

## Temporary Images

Until you add these images, the code will fall back to existing images in `/assets/img/`. Once you add the tree images with the exact filenames above, they will automatically be used instead.
